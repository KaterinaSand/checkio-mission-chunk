You have a lot of work to do, and you want to split this work on smaller pieces, so you will know on which piece you will do on Monday, which will be for Tuesday and so on.

Spit an array to an array os smaller arrays with the same size (chunks). The last chunk can be smaller than default chunk-size. If array is empty then you should not have chunks at all.

Input: Two arguments. {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %} and chunk size.

Output: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %} with chunked {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
chunkingBy([5, 4, 7, 3, 4, 5, 4], 3) == [[5, 4, 7], [3, 4, 5], [4]]
chunkingBy([3, 4, 5], 1) == [[3], [4], [5]]
{% else %} 
chunking_by([5, 4, 7, 3, 4, 5, 4], 3) == [[5, 4, 7], [3, 4, 5], [4]]
chunking_by([3, 4, 5], 1) == [[3], [4], [5]]
{% endif %}

Precondition: chunk-size > 0
